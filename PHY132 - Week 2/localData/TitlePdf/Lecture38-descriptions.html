"Double Slit Euler's Relation"
In order to integrate the electric eld for each "y" contribution, we change the electric eld from a cosine function into a
complex function using Euler's relation, to make the integral simpler to solve.
"Dening the Denominator"
Since we know we must sum each component of the electric eld over "y," Prof. Hemmick discusses how to build the
relationship between dE and dy, in order to know that we've summed over each possible portion. We use what we
determine in the denominator to set the integral.
"Determining the Bounds"
In order to complete our integral, we need to determine our bounds, setting them to ensure that we sum over every
possible location that contributes light.
"Solving the Integral"
We work through the integral that we built, now with the correct terms and bounds, and end up with quite a few terms that
can be simplied.
"Simplifying the Answer"
Because of our usage of Euler's Relation, we can simplify our result using the exponential product rule and basic rules of
distribution.
"Sinc Function & Intensity Formula"
Continuing to work our result from before, Prof. Hemmick solves for the intensity function, and gets an interesting result
which looks similar to what we calculated for a single slit, but with new terms.
"Sketching Double Slit Intensity"
Using the intensity formula we solved for, Prof. Hemmick sketches the "envelope" intensity function for the double slit.
